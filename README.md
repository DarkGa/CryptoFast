# CryptoFast

### Установка

#### Скачиваем гит

```apt install git```

#### Клонируем репозиторий

```git clone https://github.com/DarkGa/CryptoFast.git```

#### Устанавливаем зависимости

```apt install clang make```

#### Компилируем

```cd CryptoFast; make```

### Использование

##### Чтобы узнать все функции достаточно ввести команду

```cryptor -h```
